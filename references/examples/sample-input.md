# 示例输入：智能出题请求

## 出题需求

```
学科：高中数学
知识点范围：
  - 函数概念与性质（定义域、值域、单调性、奇偶性）
  - 基本初等函数（指数函数、对数函数、幂函数）
  - 导数及其应用（导数定义、求导法则、导数与单调性/极值/最值）
题型要求：
  - 单选题 × 5
  - 填空题 × 3
  - 解答题 × 2
总分：100分
难度分布：易30% (30分) | 中50% (50分) | 难20% (20分)
Bloom层次：理解30% | 应用40% | 分析20% | 评价10%
目标：高三一轮复习
考试时间：90分钟
```

## 补充要求
- 解答题需要完整的评分标准（步骤分）
- 每题标注具体考察的知识点
- 情境题可使用近2年的科技/生活素材
- 注意新高考I卷的命题风格
