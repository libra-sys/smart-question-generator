# 智能出题与评测系统

> Smart Question Generator — Bloom分类法自动出题 · 难度标定 · 智能组卷

[![Version](https://img.shields.io/badge/version-v1.0.0-blue)]()
[![Platform](https://img.shields.io/badge/platform-WorkBuddy-purple)]()

## 功能概述

基于 Bloom 认知分类法的自动化试题生成工具。指定学科知识点、难度等级和题型，自动生成含标准答案、详细解析和难度标定的试题，支持批量生成和智能组卷。

## 核心功能

| 功能 | 说明 |
|------|------|
| **多维度出题** | 知识点 + 难度等级 + 题型（选择/填空/简答/论述）组合输入 |
| **Bloom 分类标定** | 记忆 → 理解 → 应用 → 分析 → 评价 → 创造，六层认知难度标定 |
| **答案+解析** | 每道题含标准答案 + 详细解析 + 知识点标注 |
| **智能组卷** | 按题型比例、难度分布、知识点覆盖自动生成完整试卷 |

## 触发方式

```
出题 [知识点] [难度] [题型]
生成试题 [科目] [数量]
组卷 [学科]
```

## 应用场景

- 教师日常出题
- 在线题库建设
- 企业培训考核
- 标准化考试组卷
